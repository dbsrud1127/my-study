# 자바의 특징 및 장점과 단점

1.특징

-운영체제의 독립성 : Windows, macOS, Linux 등 운영체제에 관계없이 실행 가능

-객체 지향 : 모든것을 객체 중심으로 생각하고 설계해 코드 재사용과 유지보수성을 높임

-안전성 : 기존랭귀지와 비교했을 때 가비지 컬렉션(GC)을 통해 안정적인 메모리 관리가능

2.장점

-독립성: 한 번 작성하면 어디에서든 실행 가능한  WORA (Write Once, Run Anywhere)환경을 제공한다.

-객체지향 프로그래밍:  현실 세계의 객체를 모델링하여 코드를 작성할 수 있으며, 코드의 재사용성이 높고 유지보수가 용이하여 대규모 소프트웨어 개발에 적합하다.

-안정성: **C, C++ 등 기존 언어와 달리 가비지 컬렉션(GC)을 통해 메모리를 자동 관리해 오류 발생 가능성을 줄이고 안정적인 프로그램 실행을 지원한다.**

3.단점

-실행 환경 의존성: 자바 프로그램을 실행하기 위해 JVM(자바 가상 머신, Java Virtual Machine) 이 설치되어 있어야 한다.
